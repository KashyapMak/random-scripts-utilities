# Local JSON Compare Tool (Revised)

## Changes in this version
- Comparison area is hidden by default.
- After clicking **Compare**, the editor inputs are hidden and only the comparison view is shown.
- Added **Back to Editors** button.
- Synchronized vertical and horizontal scrolling between both comparison panes.
- Wider layout to use browser width better for long JSON lines.
- Comparison tables support horizontal scrolling for long content.

## Files
- `index.html`
- `styles.css`
- `app.js`
- `README.md`

## How to use
1. Extract the zip.
2. Open `index.html` in your browser.
3. Paste or load JSON into both editors.
4. Click **Compare**.
5. Use **Back to Editors** if you want to modify the inputs again.
