# JSON Schema Generation & Validation Tool

## Changes in this version
- Tool name updated to **JSON Schema Generation & Validation Tool**.
- Added tabs: **Inputs** and **Validation Results**.
- By default, only the **Inputs** view is shown.
- **Schema Validation Summary** and **JSON Preview with Invalid Parts Highlighted** are shown only after clicking **Validate JSON** with valid left-side JSON and right-side schema.
- Layout uses more browser width for working with long JSON and schema content.

## How to use
1. Open `index.html` in a browser.
2. Keep JSON on the left and schema on the right.
3. Click **Generate Schema** to create schema from the left-side JSON.
4. Click **Validate JSON** to open the **Validation Results** tab with summary, errors, and highlighted JSON preview.
